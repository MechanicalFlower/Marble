## About

Marble is an open source mini-game about marble racing.

### Controls

- `WASD` to move.
- `Mouse` to move the camera around.
- `Tab` to follow a marble.
- `ESC` to open and close pause menu.

### Features

- Name your marbles.
- Start procedural races.
- Watch the real-time ranking.
